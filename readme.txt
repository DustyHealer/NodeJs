NPM Commands

# To check the installed version of node and npm
node --version
npm --version

# To initialize a backend project in nodeJs
node init

# To install node express server for backend apps
node i express --save

# To install packages using the terminal
node install package-name
node i package-name

# To install nodemon globally - A global package will be available to all the projects
npm i -g nodemon

# To install nodemon as a dev dependency
npm install --save-dev nodemon

# To run a project using node or nodemon
node index.js or node index
nodemon index.js or nodemon index
